Déclaration de confidentialité de Etogs

Que signifie cette déclaration ?

La présente déclaration de confidentialité relative à la protection des données personnelles décrit comment Etogs utilise vos données personnelles.

Quelles données personnelles Etogs collecte-t-il ?

Nous collectons et traitons les données personnelles lorsque vous répondez à notre questionnaire. Ces dernières comprennent :

Votre Nom et Prénom ;

Votre sexe, votre âge ;

Vos coordonnées, telles que votre ville d’habitation, votre adresse e-mail (si vous l’avez renseigné dans le questionnaire) ;

Votre situation familiale et votre profession ;

Vos préférences et habitudes d’achat ;

Les informations afférentes au navigateur ou à l’appareil que vous utilisez pour accéder à notre site Web, votre adresse IP et vos préférences concernant les cookies (témoins de connexion).

Comment Etogs utilise vos données personnelles et quelle est la base juridique de cette utilisation ?

Nous utilisons les données personnelles renseignées dans notre questionnaire aux fins suivantes :

Améliorer le projet en comparant les données récoltées ;

Communiquer avec vous sur l’avancée du projet

Si vous refusez que Etogs collecte et utilise les données personnelles que vous avez renseignées dans le questionnaire :

L’utilisation de vos informations ne sera pas possible ;

Nous ne serons pas autorisés à utiliser vos réponses au questionnaire et les champs que vous avez remplis seront alors supprimés.

Lorsque vous acceptez que Etogs collecte et utilise les données personnelles que vous avez renseignées dans le questionnaire, vous nous donnez un consentement pour la réutilisation de ces données.

Pendant combien de temps Etogs va conserver vos données personnelles ?

Nous nous engageons à conserver ces données pendant une durée de 12 mois.

Etogs va-t-il partager vos données personnelles ?

Aucunes de vos données personnelles renseignées dans notre questionnaire ne seront partagée.

Nous nous engageons à garder confidentielle vos informations.

Comment pouvez-vous vous désinscrire ?

Si vous ne souhaitez plus recevoir d’emails de notre part et que nous supprimions toutes vos données personnelles, vous pouvez à tout moment formuler votre demande à cette adresse : etogs.app@gmail.com

Quels sont vos choix et vos droits ?

• De demander à Etogs une copie de vos données personnelles ;
• De demander à Etogs de corriger vos données si elles sont inexactes ou incomplètes ;
• De demander à Etogs de supprimer vos données ;
• De restreindre le traitement de vos données personnelles ;

Pour nous contacter :

Nous espérons pouvoir répondre aux questions que vous pourriez avoir sur la façon dont nous traitons vos données. Si vous avez de quelconques inquiétudes quant à la façon dont nous traitons vos données, n’hésitez pas à nous écrire à cette adresse : etogs.app@gmail.com

Qui est en charge du traitement de vos données personnelles :

Inès MOKADDEM, porteuse du projet Etogs.
